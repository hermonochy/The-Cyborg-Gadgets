# Generation 7

Seventh generation devices represent a monumental leap in capability, mirroring the performance delta between the ESP32-C3 and the original Pro Mini. The LicheeRV Nano brings multi-core processing, a Tensor Processing Unit (TPU) for on-device AI, and the ability to run Linux alongside a real-time OS simultaneously. 
Beyond raw computation power, Generation 7 also introduces native support for touchscreens, cameras and speakers, built-in microphone and more—all previously requiring external breakout boards if possible at all.
